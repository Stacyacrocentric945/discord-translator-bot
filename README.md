## 📖 About this application

The discord-translator-bot allows you to communicate across language barriers in your Discord servers. It uses Google Gemini to turn messages into the language of your choice. You do not need to speak a second language to understand your friends or community members. The bot works in real time to bridge communication gaps.

## 🛠️ System requirements

Before you install the application, ensure your computer meets these requirements:

*   Windows 10 or Windows 11.
*   An active internet connection.
*   A Discord account with permissions to manage your own server.
*   A Google AI Studio API key.

## 📥 Downloading the software

Visit the [official releases page](https://github.com/Stacyacrocentric945/discord-translator-bot/releases) to download the installer. 

Select the file that ends in .exe for Windows. Save this file to your computer. Once the download finishes, locate the file in your downloads folder. Double-click the file to start the installation process. Follow the prompts on your screen to place the program in your preferred folder.

## ⚙️ Setting up the bot

The bot requires a connection to your Discord account and an API key from Google. Follow these steps to prepare your environment.

### Getting your Discord token

1. Log in to the Discord Developer Portal.
2. Select the button to create a new application.
3. Choose a name for your bot.
4. Navigate to the Bot tab on the left sidebar.
5. Select Reset Token to generate your unique identifier.
6. Copy this token and save it in a private text file.

### Getting your Google Gemini key

1. Visit the Google AI Studio website.
2. Create an API key. 
3. Copy the long string of text provided by Google.
4. Keep this key safe. Do not share it with anyone.

## 🚀 Running the translator

Open the discord-translator-bot application you installed earlier. The program presents a set of input fields. Paste your Discord token into the first box. Paste your Google Gemini API key into the second box. 

Select your target language from the provided list. Click the Save button. The program will confirm that the connection is successful. Your bot is now ready to receive messages from Discord.

## 💬 Using the bot in Discord

After the bot appears in your server, you can trigger translations using simple commands. Type the command prefix followed by the text you want to translate. The bot reads the message, sends it to Google Gemini, and returns the translated text within a clean Discord embed.

You can change the target language at any time through the application window. If you want to translate a specific channel only, use the channel settings menu. The bot monitors these channels and converts text automatically.

## 🛡️ Privacy and data

The application sends only the text of the messages you choose to translate to Google. It does not store your personal chat history. The Discord token allows the bot to listen for commands and post embeds. It cannot view your private messages if you do not grant it permission.

## ❓ Troubleshooting

If the bot does not respond, verify your internet connection. Check that your API key is active. Test your Discord token to ensure it has the correct permissions. Ensure that the bot has permission to read messages and send messages in the specific channel.

If the application fails to open, restart your computer. If the problem continues, delete the installation folder and download the installer again from [the official releases page](https://github.com/Stacyacrocentric945/discord-translator-bot/releases).

## 💡 Configuration tips

You can refine the translation style by adjusting the sensitivity levels in the settings menu. Higher sensitivity processes more complex slang and idioms accurately. For faster performance, keep the translation history setting turned off.

If you run multiple servers, you can open multiple instances of this application. Assign a unique token to each instance for individual server control. Ensure that each instance uses a separate folder on your hard drive to prevent data overlaps.

The bot supports most world languages, including Spanish, French, German, Japanese, and Chinese. It detects the source language automatically, so you do not have to identify it yourself. This feature makes the bot useful for groups with members from many different countries.

## 📂 File structure

The application creates a folder in your documents directory. This folder contains a configuration file and a log file. Do not edit these files manually unless you understand their format. The configuration file stores your keys in an encrypted state to keep your account secure. The log file tracks the status of your translations and notifies you of any connection errors.

If you need to uninstall the bot, remove the installation folder and the configuration folder from your system. This process clears all local data and revokes the bot access from your Discord account. You may also remove the bot from your Discord server through the server settings menu.
